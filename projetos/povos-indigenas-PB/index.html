Sobre o projeto

Este projeto apresenta informações históricas e culturais sobre diferentes povos indígenas relacionados ao território paraibano e nordestino.

O objetivo é contribuir para a valorização da diversidade indígena brasileira, apresentando informações baseadas em pesquisas históricas, arqueológicas e referências institucionais.

Importante sobre as imagens

As imagens utilizadas neste projeto possuem caráter meramente ilustrativo.

Devido à ausência de registros visuais históricos de alguns povos indígenas, foram utilizadas imagens e representações relacionadas ao contexto indígena brasileiro apenas como apoio visual.

As imagens apresentadas não representam necessariamente os povos descritos nas páginas.

Objetivo

Este site foi desenvolvido como projeto acadêmico para a Semana Acadêmica Unificada (SAU) do IFPB Campus Itabaiana.

Além da apresentação histórica, o projeto busca utilizar tecnologias web para divulgar conhecimento sobre a cultura indígena brasileira.

Referências gerais

As informações foram construídas a partir de fontes como pesquisas acadêmicas, documentos institucionais e materiais educativos relacionados aos povos indígenas brasileiros.
